# over35-blacklist-china

中文介绍

本项目用于整理并公开中国地区明确或隐性不招聘 35 岁及以上求职者的企业名单，旨在记录职场年龄歧视现象，为 35+ 职场人求职避坑提供参考，推动职场年龄平等与就业公平。


English Description

A list of companies in China that explicitly or implicitly refuse to hire people aged 35 and above.
This project aims to record age discrimination in the workplace, help job seekers over 35 avoid unfair employers, and promote age equality in employment.
